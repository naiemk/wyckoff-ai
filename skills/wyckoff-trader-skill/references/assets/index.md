# Local Source Assets

These assets are bundled into the skill so future use does not require browsing, scraping, or URL parsing.

## Available Local Corpora

- `book/`: simplified local text version of the 248-page Wyckoff book.
- `crypto_archive/`: simplified local text version of the 46 captured Wyckoff crypto posts.
- `bruce_fraser_stockcharts/`: simplified local text version of 243 Bruce Fraser StockCharts articles from May 11, 2015 through March 1, 2026.

## Suggested Loading Strategy

- For formal Wyckoff framework questions, start with `book/index.md` then `book/full_text.txt` or relevant `book/page_*.txt` files.
- For crypto adaptation questions, start with `crypto_archive/index.md` then load only the relevant posts from `crypto_archive/posts/`.
- For applied Wyckoff case studies, Point & Figure technique, campaign logic, trend-stride work, and top-down stock selection, start with `bruce_fraser_stockcharts/index.md`.
- Prefer these local assets over external URLs.
